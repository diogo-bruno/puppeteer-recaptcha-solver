# Running

```
git clone https://github.com/diogo-bruno/puppeteer-recaptcha-solver.git
cd puppeteer-recaptcha-solver

-------------------
npm install
npm start
-------- OR -------
docker-compose up
```

# Test

```
npm run test-http-resolve-recaptcha
```
